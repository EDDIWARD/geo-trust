# 农产品经营策略 RAG 原素材库

这个目录用于存放面向商家经营决策的原始资料，优先保存一手原文、官方页面、白皮书、学术论文和企业公开材料。

目录说明：

- `00_资料清单`：资料总表、来源状态、用途说明
- `01_品牌定位与品牌化`：品牌定位、区域公用品牌、品牌建设方法
- `02_定价分级与经营策略`：分级定价、价格带、经营模型
- `03_包装礼盒与消费场景`：包装规格、礼盒化、消费场景设计
- `04_区域市场与渠道`：区域流向、城市偏好、电商与直播渠道
- `05_溯源信任与购买行为`：溯源、消费者信任、购买意愿
- `06_案例与复盘`：成功案例、失败案例、跨区域经营打法
- `99_待核验`：暂时无法确认原始出处或仅有二手转载的材料

保存原则：

1. 优先保留原始链接和原始文件。
2. 同时记录来源类型：官方 / 学术 / 企业公开 / 行业媒体 / 二次转载。
3. 不能确认出处的内容先放入 `99_待核验`，不直接作为核心知识源。
